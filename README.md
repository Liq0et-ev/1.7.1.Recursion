# Ņūtona algoritms kvadrātsaknes aprēķināšanai 

Šī Java programma implementē Ņūtona  iteratīvo metodi kvadrātsaknes aprēķināšanai dotam skaitlim. <br>
Programma sāk ar sākotnējo tuvinājumu un atkārtoti uzlabo rezultātu, līdz tiek sasniegta noteiktā precizitāte .<br> 
Rezultātā tā izvada aprēķināto √a vērtību.<br>
Nepareizas ievades gadījumā  tiek izvadīts kļūdas paziņojums.<br>
